Если при запуске ItemChecker выходит ошибка 403 Forbiden или 304 Not Modified

1. Нажимаешь "ОК"
	1.1. Лучше немного подождать минут 1-2.
2. Перезапускаешь.
3. И так пока не запуститься.

====================================================================================

Если при запуске ItemChecker просит установить .NET 7.x.x

1. Открываешь ссылку ".NET 7 Desktop Runtime"
2. Скачиваешь установчик из ".NET Desktop Runtime 7.x.x" -> "Windows x64"
3. Перезапускаешь.

************************************************************************************

If you get a 403 Forbiden or 304 Not Modified error when running ItemChecker

1. Click "OK"
	1.1. It is better to wait a little 1-2 minutes.
2. Restart.
3. And so on until it starts.

==================================================================================

If ItemChecker asks to install .NET 7.x.x when starting

1. Open the link ".NET 7 Desktop Runtime"
2. Download the installer from ".NET Desktop Runtime 7.x.x" -> "Windows x64"
3. Restart.
